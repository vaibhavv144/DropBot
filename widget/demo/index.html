Widget demo

This page simulates a customer website with your chat widget in the corner. Paste your bot's public key from the dashboard Embed tab and click Load widget.

Tip: open with a pre-filled key — ?bot=pub_xxx&api=http://127.0.0.1:8000

If chat returns 403, set the bot's Allowed domains to * or add localhost in the dashboard Appearance tab.

Try asking about content you indexed:
  • A question from your crawled website
  • Something from an uploaded PDF or document

<script src="https://cdn.yourbot.com/embed.js"
        data-bot-id="YOUR_KEY"
